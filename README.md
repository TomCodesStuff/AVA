# AVA (Algorithms Visualised Amazingly)

Welcome to AVA, an algorithms visualiser designed to support array searching, array sorting and graph traversal. <br>
Not only does AVA enable customising arrays but also includes physics-based graph rendering to encourage fun, experimentation and interactivity. <br>

Want to add custom algorithms? A clear step-by-step guide is included so algorithm creation and visualisation is as easy as possible. <br>
See below for examples of what AVA can do and how to make your own algorithms. <br>

Created by Tom. I hope you enjoy AVA as much as I enjoyed making it.
